# 公众号图片素材库 - 使用说明

## 目录结构
```
公众号图片素材库/
├── 00_待整理/           # 所有新图片先丢这里
├── 01_人物人设/         # 律师形象照、职业照、办公、开会、看材料
├── 02_办公场景/         # 办公桌、电脑、文件、律所环境、会议室
├── 03_案件表达/         # 合同、聊天记录、工资条、证据照片、法院环境
├── 04_生活化素材/       # 喝茶、走路、看书、长沙街景、日常感受
├── 05_已发布/           # 公众号已用过的图（复制一份）
├── 06_专题素材包/       # 按业务专题分类
│   ├── 拖欠工资奖金/
│   ├── 违法辞退/
│   ├── 竞业限制/
│   ├── 调岗降薪/
│   └── 高管离职谈赔/
├── 素材登记表.csv       # 素材管理表格
└── README.md           # 本说明文件
```

## 命名规则
**格式：** `日期+类别+关键词+序号`

**示例：**
- `2026-04-06_人物人设_办公室看材料_01`
- `2026-04-06_办公场景_会议室电脑桌_02`
- `2026-04-06_案件表达_劳动合同打码_01`
- `2026-04-06_生活化素材_长沙街景夜色_03`

**原则：** 文件名要写成“搜这个词就能出来”

## 使用流程

### 平时
- 手机拍完先不管
- 一周统一导出一次

### 每周整理（30-60分钟）
1. 新图丢进 `00_待整理`
2. 快速删废片
3. 留下能用的
4. 按规则改名
5. 放进对应文件夹
6. 在登记表补一行

### 写文章时找图
1. 确定文章类型
2. 按逻辑找图：
   - **高管被架空辞退** → 案件表达 + 冲突感 + 办公室/会议室
   - **律师观察判断** → 人物人设 + 专业感 + 办公室  
   - **情绪共鸣内容** → 生活化素材 + 松弛感/压迫感 + 长沙街景

## 专题素材包
这是最值钱的部分，按高频业务建专题：
- 每个专题放最常用配图
- 已打码证据图
- 适合封面的图
- 以前发过效果好的图

## 避坑指南
1. **别一开始全量整理** → 先整理最近300张
2. **别分类太细** → 越细越没人维护
3. **别混放已发布/未发布** → 复盘会乱套

## 下一步
1. 把最近一批图片丢进 `00_待整理`
2. 挑出50张最可能用于公众号的图
3. 按命名规则改名，放进对应文件夹
4. 在登记表里记录

完成这轮，素材库就活了。
