# 飞书多维表素材库模板字段

## 表名
`公众号图片素材库`

## 字段配置

### 1. 图片文件名（文本）
- **字段名：** 图片文件名
- **说明：** 完整的文件名，如 `2026-04-06_人物人设_办公室看材料_01`
- **必填：** 是

### 2. 所在文件夹（单选）
- **字段名：** 所在文件夹
- **选项：**
  - 00_待整理
  - 01_人物人设
  - 02_办公场景
  - 03_案件表达
  - 04_生活化素材
  - 05_已发布
- **必填：** 是

### 3. 场景（多选）
- **字段名：** 场景
- **选项：**
  - 办公室
  - 会议室
  - 法院
  - 路上
  - 咖啡馆
  - 长沙街景
- **必填：** 否

### 4. 用途（多选）
- **字段名：** 用途
- **选项：**
  - 封面
  - 配图
  - 人设
  - 案例说明
  - 证据展示
- **必填：** 否

### 5. 情绪（多选）
- **字段名：** 情绪
- **选项：**
  - 专业
  - 压迫感
  - 冲突感
  - 松弛感
  - 信任感
- **必填：** 否

### 6. 关键词（文本）
- **字段名：** 关键词
- **说明：** 用顿号分隔，如 `律师、阅读材料、劳动法`
- **必填：** 否

### 7. 是否已发布（单选）
- **字段名：** 是否已发布
- **选项：**
  - 是
  - 否
- **默认值：** 否
- **必填：** 是

### 8. 适合什么选题（文本）
- **字段名：** 适合什么选题
- **说明：** 用顿号分隔，如 `竞业限制、离职谈赔、拖欠工资`
- **必填：** 否

### 9. 备注（文本）
- **字段名：** 备注
- **说明：** 补充信息，如 `横图，适合公众号首图下第二张`
- **必填：** 否

### 10. 图片链接（附件）
- **字段名：** 图片
- **说明：** 上传图片文件
- **必填：** 否

### 11. 创建时间（创建时间）
- **字段名：** 创建时间
- **自动生成**
- **必填：** 否

### 12. 更新时间（修改时间）
- **字段名：** 更新时间
- **自动生成**
- **必填：** 否

## 视图建议

### 1. 按文件夹分组视图
- **分组字段：** 所在文件夹
- **排序：** 创建时间倒序
- **用途：** 快速查看各文件夹下的图片

### 2. 待整理视图
- **筛选条件：** 所在文件夹 = 00_待整理
- **排序：** 创建时间正序
- **用途：** 每周整理时使用

### 3. 已发布视图
- **筛选条件：** 是否已发布 = 是
- **排序：** 创建时间倒序
- **用途：** 查看已发布图片，方便复盘

### 4. 专题查找视图
- **分组字段：** 适合什么选题（可按文本包含筛选）
- **排序：** 创建时间倒序
- **用途：** 写文章时快速找专题图片

## 使用流程

### 新增图片
1. 在 `00_待整理` 文件夹新增一行
2. 填写图片文件名
3. 上传图片文件
4. 其他字段可先不填

### 每周整理
1. 打开"待整理视图"
2. 逐条审核图片
3. 更新字段：
   - 修改"所在文件夹"
   - 添加场景、用途、情绪标签
   - 填写关键词和适合选题
   - 添加备注

### 发布后更新
1. 图片发布后
2. 将"是否已发布"改为"是"
3. 可考虑复制图片到 `05_已发布` 文件夹（新建一行）

## 高级功能（可选）

### 1. 关联字段
可创建"文章记录表"，关联已发布的图片，记录：
- 文章标题
- 发布时间
- 阅读量
- 图片使用效果

### 2. 公式字段
- **图片类型：** 根据文件名自动提取类型（人物、场景、案件等）
- **使用次数：** 统计图片被关联到文章的次数

### 3. 自动化
- 图片上传后自动提取文件名中的日期
- 定期提醒整理待处理图片

## 导入现有数据
如果你已经有 `素材登记表.csv`，可以直接导入：
1. 在飞书多维表点击"导入"
2. 选择CSV文件
3. 映射字段对应关系
4. 导入后检查数据
