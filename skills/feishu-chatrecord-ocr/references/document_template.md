# 委托聊天记录OCR输出模板

## 文档命名

「**来源-日期-当事人**」

示例：`A-26/4/7-许亮`

### 来源编码说明

| 编码 | 来源 |
|------|------|
| A | 抖音 |
| B | 视频号 |
| E | 其他 |

## 对话内容格式

每条消息格式：`[日期时间] 当事人/律师：内容`

示例：
```
[2026年4月7日 10:30] 当事人：律师您好，我想咨询一下劳动纠纷的事
[2026年4月7日 10:31] 律师：您好，请说
[2026年4月7日 10:32] 当事人：我是公司这边拖欠工资
[2026年4月7日 10:33] 律师：拖欠多久了？涉及多少人？
```

## 完整整理原则

**必须提取所有独立案例，不得遗漏。**

判断独立案例的标准：
- 不同当事人 = 不同案例
- 同一当事人不同纠纷事项 = 不同案例
- 同一当事人相同事项不同时间 = 同一案例

## 日期格式规范

统一使用：`26/4/7`（年月日，不补零）

## 原始对话逐字输出要求

- 不提炼、不总结、不删改
- 错别字可适当修正
- 逐条呈现，不合并
