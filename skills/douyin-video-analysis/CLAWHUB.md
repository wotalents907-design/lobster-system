# ClawHub Listing Notes / ClawHub 上架说明

## Short description / 短描述
Douyin video analysis pipeline for OpenClaw: capture metadata and audio, transcribe speech, summarize content, extract useful information plus evidence, critically analyze the script, and save everything into Obsidian.

用于 OpenClaw 的抖音视频分析流水线：抓取页面信息与音频、转写口播、总结内容、提取有用信息与对应证据、进行批判性思维分析，并将结果写入 Obsidian。

## Key capabilities / 核心能力
- Resolve Douyin short/full URLs / 解析抖音短链与完整链接
- Extract title, metadata, visible text / 抓标题、元信息、可见文字
- Capture and download audio / 提取并下载音频
- Transcribe audio locally / 本地音频转写
- Create transcript notes / 生成转写笔记
- Create structured analysis notes / 生成结构化分析笔记
- Judge AI-assisted likelihood / 判断是否像 AI 辅助文案
- Perform critical-thinking analysis / 做批判性思维分析
- Save results into Obsidian / 保存到 Obsidian

## Recommended audience / 适合谁
- researchers / 研究短视频内容的人
- creators / 做内容创作的人
- operators / 做内容运营的人
- anyone who wants more than surface-level hot takes / 不满足于表层热评的人

## Important message / 重要说明
This skill is designed to avoid shallow commentary.
It always tries to:
1. summarize first
2. extract useful information and evidence second
3. critique third

这个 skill 专门用来避免“上来就嘴炮”。
它会优先：
1. 先总结
2. 再提取有用信息与证据
3. 最后批判性分析
