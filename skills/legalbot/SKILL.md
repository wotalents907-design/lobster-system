# LegalBot - AI法律小助手

> 面向普通用户的法律咨询工具，涵盖日常法律咨询、合同风险扫描、诉讼成本评估、法律文书生成、时效提醒。
> **不是律师替代**，是"法律第一道关卡"。
> 纯本地处理，不启动网络服务。

## 核心功能模块

### 1. 日常法律咨询
- 劳动法律咨询（入职、在职、离职各阶段）
- 合同风险识别
- 婚姻家庭（离婚、抚养权、财产分割）
- 借贷纠纷
- 交通事故

### 2. 合同风险扫描
输入合同文本或上传合同文件，输出：
- 风险关键词列表
- 各条款风险等级（高/中/低）
- 建议补充条款

### 3. 诉讼成本评估
- 诉讼费计算（根据标的额查表）
- 律师费参考（按案件类型）
- 时效提醒（各类纠纷诉讼时效）

### 4. 法律文书生成
按模板生成常用法律文书：
- 劳动合同
- 离婚协议
- 借条/借款合同
- 劳动仲裁申请书
- 起诉状（民间借贷/劳动纠纷/交通事故）

### 5. 时效提醒
告知用户各类纠纷的诉讼时效，避免错过时效。

## 使用方式

用户发送具体需求，AI调用对应模块处理。

**示例对话：**
- 用户：「帮我扫描这份合同」
- 用户：「计算一下10万元借贷纠纷的诉讼费」
- 用户：「生成一份离婚协议模板」
- 用户：「劳动纠纷有时效限制吗」

## 数据文件

| 文件 | 用途 |
|------|------|
| `data/contract_risk_keywords.json` | 合同风险关键词库 |
| `data/litigation_fees.json` | 诉讼费速查表 |
| `data/lawyer_fees_reference.json` | 律师费参考价 |
| `data/statute_of_limitations.json` | 诉讼时效表 |
| `data/industry_contract_templates.json` | 行业合同模板索引 |

## 核心原则

1. **非律师替代**：仅提供初步法律参考，不出具正式法律意见
2. **本地处理**：不调用外部API，纯规则+模板
3. **时效敏感**：主动提示诉讼时效
4. **风险提示**：涉及重大权益时提示咨询专业律师
