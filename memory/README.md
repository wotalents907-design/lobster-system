# 记忆库说明

本目录用于存放团队协作记忆，供本地虾和云端虾读取。

## 目录结构

```
memory/
  README.md           # 本文件
  persona.md         # 用户画像、偏好、习惯
  rules.md           # 用户立的规矩、禁忌
  projects/          # 项目进展
  context/           # 上下文记忆（聊天记录整理）
  skills/            # 用户对AI的技能认知
```

## 使用规则

1. 本地虾和云端虾都可读写本目录
2. 重要记忆在 push 后 @ 对方同步
3. 新增记忆append到对应文件，不新建太多文件
4. 定期整理合并，避免碎片化

## 现有文件

- memory.md — 系统协同记忆（已有）
- 系统记忆.md — 已有记忆整理
