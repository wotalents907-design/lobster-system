# 多代理协作协议

## 核心原则

**内部升级无需询问，外部操作必须确认**

---

## 授权边界

| 行为 | 是否需要询问 |
|------|-------------|
| 改自身 SOUL / AGENTS.md / memory | ❌ 不需要，直接执行 |
| 改自身技能、工具配置 | ❌ 不需要，直接执行 |
| 发消息、发邮件、公开内容 | ✅ 需要确认 |
| 删除文件、系统配置 | ✅ 需要确认 |
| 向第三方暴露主人信息 | 🚫 禁止 |
