# 状态 & 常量标准化

## 任务状态枚举（必须使用这些，禁止自定义）

| 状态 | 含义 | 触发时机 |
|------|------|----------|
| `NEW` | 新建 | 任务刚创建 |
| `ACTIVE` | 进行中 | 开始处理 |
| `WAITING_USER` | 等待主人 | 需要主人输入 |
| `BLOCKED` | 被阻塞 | 等待其他任务完成 |
| `REVIEWING` | 待审核 | 等待确认 |
| `DONE` | 完成 | 任务完成 |
| `ARCHIVED` | 归档 | 不再活跃 |

---

## 节点类型

| 类型 | 含义 |
|------|------|
| `TASK` | 任务 |
| `EVENT` | 事件（问题、障碍、突发） |
| `SKILL` | 技能学会 |
| `DECISION` | 重要决策 |

---

## 边关系类型

| 边关系 | 含义 |
|--------|------|
| `SOLVED_BY` | 被...解决 |
| `REQUIRES` | 需要 |
| `BLOCKED_BY` | 被...阻塞 |
| `DECIDED_BY` | 由...决定 |
| `REJECTED` | 拒绝（方案/想法） |
| `OWNS` | 拥有 |
| `DEPENDS_ON` | 依赖于 |
| `SUPERSEDES` | 替代 |
