# 规则 / 任务 / 记忆 / 日志 四类文件分工说明（V1.0）

## 一、总原则

四类文件各管一类事，不允许混写。

- 规则：写“以后都要这样做”的制度
- 任务：写“当前要做什么、做到哪了”
- 记忆：写“长期有用、后面还要记得的事实”
- 日志：写“执行过程中产生的临时过程信息”

## 二、规则文件

规则文件只写长期适用的执行标准、行为约束、协同制度、汇报要求、任务执行铁律、文件使用规则。

## 三、任务文件

任务文件只写当前有哪些任务、任务在哪个状态、谁在执行、任务结果、任务阻塞原因。

## 四、记忆文件

记忆文件只写长期有效的重要事实、稳定偏好、重要身份信息、后续可能反复用到的关键结论、阶段性重大决策。

## 五、日志文件

日志文件只写执行过程、命令输出、阶段排障痕迹、临时观察结果、过程型心跳记录。

## 六、一句话总结

**规则决定以后怎么做，任务决定现在做什么，记忆决定以后记住什么，日志保留这次是怎么做的。**
