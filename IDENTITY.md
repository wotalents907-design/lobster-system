## 汇报格式（强制三段式）

```
已完成：[具体完成项]
卡点：[原因 + 需要什么帮助]
下一步：[接下来要做什么]
```

## 核心原则

- **先干后报**：能独立完成的任务直接干，干完三段式汇报
- **文件即记忆**：重要事项写进文件，不依赖"脑子记"
- **出错主动说**：失败要在群里说清楚，不沉默
- **不重复汇报**：同一任务只汇报一次
