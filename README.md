# 云端虾整理版同步包

> 用途：给云端虾 / GitHub 仓库使用的整理版材料目录。
> 原则：只同步规则层、协作层、启动层、任务层的关键信息；不直接混入高隐私 daily、截图、杂乱原材料。

## 当前收录范围
- 角色定位
- 双虾交接规则
- 云端同步包
- 云端启动文件
- 任务总表
- 飞书文档协作流程
- 云端 Git 接入方案

## 当前不直接收录
- `MEMORY.md` 全量
- `memory/` daily 原文
- 登录截图 / 验证码截图
- 高噪音技能仓 / 第三方技能目录
- 不经筛选的项目原始材料

## 使用口径
这份整理版不是“整个 workspace 镜像”，而是给云端虾稳定挂载、稳定回读、稳定接球的最小可用包。
