# 公众号双号内容系统策略

## 一、系统目标

本系统服务两个公众号：

1. 客户服务号：面向高净值打工人 / 中高管 / 离职赔偿与劳动纠纷客户
2. 同行订阅号：面向律师同行 / 潜在合作方 / 未来团队与行业影响力对象

目标不是单纯积累文章，而是形成“可持续规划、可跟踪推进、可复用沉淀、可程序化维护”的内容系统。

## 二、双号定位

### 1. 客户服务号
- 角色：转化阵地
- 目标：建立信任、筛选咨询、承接成交
- 核心价值：让目标客户相信“你能处理我的问题，而且处理得值钱”

### 2. 同行订阅号
- 角色：影响力阵地
- 目标：建立认知优势、吸引同行、服务未来合作与团队扩张
- 核心价值：让同行相信“你有方法、有认知、有增长路径”

## 三、统一底层人设

统一人设：实战派劳动律师，既能解决具体劳动纠纷，也在思考律师业务增长与组织化路径。

要求：双号内容必须同根同源，不能人格分裂。

## 四、内容分工

### 客户服务号内容支柱
1. 真实案件拆解
2. 风险预警
3. 实操指南
4. 高价值问答
5. 信任型内容

### 同行订阅号内容支柱
1. 律师IP打法
2. 获客与转化
3. 律所/团队模型
4. 行业判断
5. 操盘日志

## 五、执行原则

1. 一篇内容只服务一个对象
2. 客户号重“转化与承接”，不是重“更新频率”
3. 同行号重“观点与势能”，不是重“普法答疑”
4. 所有内容都要能进入固定归档系统
5. 后续新增程序必须优先服务“持续维护”，不是追求花哨

## 六、阶段目标

### 第一阶段
- 先完成双号定位、栏目、选题池、模板、数据表、脚本骨架
- 建立最小可运行内容中控台

### 第二阶段
- 加入选题路由、周计划生成、状态追踪

### 第三阶段
- 加入草稿辅助、复盘归档、发布前检查
