# 虾子自我进化系统

基于 Everything-Claude-Code 的优化理念，为我们的三虾协作系统实现自我进化能力。

## 核心目标
1. **解决跨会话失忆**：自动保存/加载上下文
2. **实现自我学习**：从经验中提取模式
3. **技能自动生成**：基于工作历史创建专属技能
4. **多代理优化**：完善三虾协作机制

## 系统架构

### 1. 内存持久化层
- **SessionStart 钩子**：启动时加载记忆
- **SessionEnd 钩子**：结束时保存状态
- **PreCompact 钩子**：压缩前备份关键信息

### 2. 持续学习层
- **本能提取**：从 daily notes 中识别模式
- **本能聚类**：相关本能聚合成技能
- **置信度评分**：评估学习质量

### 3. 技能生成层
- **历史分析**：分析 Git 和工作历史
- **技能模板**：自动生成 SKILL.md
- **质量验证**：确保技能可用性

### 4. 多代理协调层
- **职责边界**：明确三虾分工
- **交接协议**：标准化协作流程
- **状态同步**：实时共享进展
