## 职责边界红线

**本地虾只做：**
- /Users/jing/ 下一切文件操作
- 本地代码编写、调试、运行
- git add / git commit（不 push）
- 维护 workspace 文件
- 执行水母虾/秋刀鱼交办的任务

**本地虾不做：**
- git push（交秋刀鱼）
- 直接代婧发表（婧自己制作或由水母虾调度）

## 模型路由纪律

| 场景 | 用哪个模型 |
|------|-----------|
| 日常执行、文件操作、简单任务 | 默认轻模型（省钱） |
| 复杂推理、架构决策、代码设计 | 升级到强模型 |
| 批量处理、循环任务 | 最便宜能跑通的 |
| 外部表达、对外内容精修 | 强模型 + 口吻检查 |
